## Key Learnings

1. **Session cookies expire naturally** — li_at from 6:08 PM invalid by 6:50 PM. This is expected security behavior.
2. **Backend should handle credentials** — SDR bot should never ask user for li_at. Fixed by putting session file on backend.
3. **Relative paths matter** — Script paths from different directories require careful relative path management (../ vs ./)
4. **Array vs Object formats** — Playwright requires cookies as array [{name, value, ...}], not flat object.
5. **Sequential > Parallel for user-facing content** — Users prefer reviewing one piece, approving, then getting next piece.
