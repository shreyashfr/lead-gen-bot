## SDR Bot — Architecture Overhaul (2026-03-21 morning)

### Final Pipeline Architecture

**Signal 1 — HIRING:**
Single command: `bash scan_hiring.sh --industry X --location Y --role Z --seniority W --max N`
Runs TWO pipelines in parallel:
- Pipeline A: `step1_yc.py --hiring` → YC founders with LinkedIn (~2 min, fast)
- Pipeline B: `step1_fetch_jobs.js` → `step2_salesnav.js` → LinkedIn jobs + Sales Nav DMs (~7-10 min, streaming)
Merges + deduplicates by LinkedIn URL into `output/leads_hiring_<ts>.json`

**Signal 2 — RECENTLY FUNDED:**
Single command: `python3 step1_yc.py --industry X --batches "X26,W26,..." --location Y --max N`
YC API → company list → scrape YC company pages for founder LinkedIn URLs directly
