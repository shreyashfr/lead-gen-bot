# Twitter/X Format Reference

## Single Tweet
- Max 260 chars
- Hook or hot take — must work standalone
- No em dashes, no semicolons
- End without a period sometimes (feels more human)

## Thread Structure
Tweet 1 — Hook (same energy as LinkedIn hook, compressed)
Tweet 2 — Setup / context
Tweets 3-6 — One point per tweet. Build the argument.
Tweet 7 — The turn or insight
Tweet 8 — Landing. Callback to tweet 1.

## Thread Rules
- Each tweet must be readable on its own
- No "1/" numbering unless thread is 6+ tweets
- Don't pad. If it can be 5 tweets, make it 5.
- Last tweet: sharp, no "follow me for more"

## Good Single Tweet Example
```
I got my first AI/ML job on a 4GB RAM PC.

Your specs aren't the problem.
```

## Good Thread Hook Example
```
Most students who can't break into AI/ML have one thing in common.

It's not their skills. It's not their degree. It's not their laptop.

```
