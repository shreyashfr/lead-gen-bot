# Instagram Carousel Format Reference

## Structure
- Slide 1: Hook (big, bold text — one line)
- Slides 2-6: One point per slide, 10-15 words max
- Last slide: Landing statement or soft CTA

## Rules
- Each slide must be readable in under 3 seconds
- No paragraphs on slides — short punchy lines only
- Visual rhythm: short → slightly longer → short → landing
- Slide 2 is the "keep swiping" moment — make it curious

## Example Carousel (4GB RAM post)
```
Slide 1: I got my first AI job on a 4GB RAM PC.

Slide 2: My family had just lost everything.

Slide 3: Dad's stock market loss. Relatives stopped calling. Mom crying.

Slide 4: I opened my laptop and Googled: "how to earn money the hard way"

Slide 5: Months of Python. 20-minute training runs. Zero support.

Slide 6: $600/week AI job. Before I turned 18.

Slide 7: Your specs aren't the bottleneck. You are.
```

## Output format when producing carousel
Label each slide clearly:
```
**Slide 1:** [text]
**Slide 2:** [text]
...
```
Also include image direction note per slide if visual context helps.
