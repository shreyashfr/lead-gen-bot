Vibe Coding changed how people build. Vibe Outreach is doing the same to how people sell.

Naval's thesis was simple. You don't write code anymore. You describe what you want, the AI builds it. The PM role just became everyone's job.

That same shift is happening in outreach.

Old playbook: hire SDRs, write sequences by hand, A/B test subject lines, watch accounts get restricted. Rinse and repeat every quarter.

New playbook: describe your ICP, set the tone, pick the channel. Hit run. The campaign runs itself.

That's what we built at Sbl.so.

You write the prompt. Sbl handles the rest... personalized messages, smart followups, real human profiles on the sends so accounts don't get flagged. Conversation automation, end-to-end.

The input is a sentence. The output is pipeline.

Vibe Outreach isn't a category yet. But it's becoming one fast.
